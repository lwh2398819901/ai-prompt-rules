**代码风格标准化决策**：
- **现状认知**：承认自己有一些不好的编程习惯
- **改进决心**：决定不管C++还是Python都遵循Google标准
- **选择原因**：Google Style Guide具有权威性和广泛认可度
- **实施范围**：
  - **后端语言**：
    - C++：Google C++ Style Guide
    - Python：Google Python Style Guide (基于PEP 8扩展)
  - **前端技术**：
    - HTML：W3C标准 + HTML5语义化最佳实践
    - CSS：BEM命名规范 + CSS模块化
    - JavaScript：Airbnb JavaScript Style Guide
    - Vue3：Vue官方风格指南 + TypeScript集成

- **监督机制**：要求AI助手在开发过程中监督代码风格，及时提醒偏差

- **技术指导需求**：在开发过程中根据项目上下文，及时推荐更好的框架或方式

- **当前AI工具**：Cursor IDE、Claude Code等

  - **架构设计AI**：系统设计和技术选型

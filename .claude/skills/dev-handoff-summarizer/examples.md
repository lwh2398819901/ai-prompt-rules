# 开发交接技能使用示例

## 示例 1：直接生成当前项目交接文档

```text
请使用 dev-handoff-summarizer 技能，为当前项目生成开发交接文档。
```

## 示例 2：围绕指定模块生成交接文档

```text
请使用 dev-handoff-summarizer 技能，围绕用户认证模块生成交接文档，默认先搜索相关代码和文档。
```

## 示例 3：跳过搜索，直接根据用户说明整理

```text
请使用 dev-handoff-summarizer 技能，不要搜索项目，直接根据我下面的说明生成交接文档。
```

## 示例 4：生成后便于另一台电脑继续开发

```text
我准备明天在公司继续开发，请使用 dev-handoff-summarizer 技能，总结当前进度，并生成 AI_HANDOFF.md 和历史归档文档。
```

## 示例 5：聚焦下一步动作

```text
请使用 dev-handoff-summarizer 技能，不要泛泛总结，重点输出当前做到哪里、下一步最小可执行动作、风险和优先阅读文件。
```

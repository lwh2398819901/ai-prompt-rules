# AI 提示词规则仓库

一套用于规范 AI 编程助手行为的提示词规则集合。

## 目录结构

| 目录 | 内容 | 用途 |
|-----|------|------|
| `CLAUDE/` | AI助手核心规则 | 定义AI工作流、行为准则 |
| `代码审查规则/` | 代码审查规则体系 | 代码质量标准、重复检测 |
| `文档编写规则/` | 文档编写规范 | AI驱动开发的文档标准 |

## 核心内容

### AI助手核心规则 (`CLAUDE/CLAUDE.md`)
- **三阶段工作流**：分析问题 → 制定方案 → 执行方案
- **八荣八耻**：AI编程行为准则（如"以暗猜接口为耻，以认真查阅为荣"）
- **用户能力档案**：个性化技术能力配置

### 代码审查规则
- **审查分级**：L1快速扫描 / L2重点审查 / L3全面审查
- **重复代码识别**：3次以上才提取的原则
- **质量标准**：函数长度、复杂度、命名规范

### 文档编写规则
- **信息密度优先**：表格 > 列表 > 段落
- **单一职责**：每个文档有唯一明确用途
- **完全自洽**：AI读一个文档就能独立工作

## 使用方式

将相关规则文件内容添加到 AI 助手的系统提示词或项目配置中。
